## 🖥️ How to Use

1. Run the Streamlit app
2. Enter customer details in the input fields
3. Click the **Predict** button
4. The system will display whether the customer is likely to churn or not

---

## 📊 Example Input Features

- Tenure
- Contract Type
- Monthly Charges
- Total Charges
- Internet Service
- Online Security
- Payment Method
- Partner
- Dependents
